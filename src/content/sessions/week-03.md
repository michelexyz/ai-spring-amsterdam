Turns the abstract safety problems from "Concrete Problems in AI Safety" into a concrete test suite: a set of reinforcement learning environments where agents can exhibit (or avoid) unsafe behaviors like reward hacking, side effects, and unsafe exploration. A hands-on framework for measuring whether an agent is actually safe.
